## 重点展开

### 这个 Claude Code 生态正在快速成熟

今天最值得关注的是 `forrestchang/andrej-karpathy-skills`（7959 stars）和 `thedotmack/claude-mem`（1897 stars）这两条。前者把 Karpathy 观察 LLM 编程陷阱的经验整理成 CLAUDE.md，后者则直接做了一个插件来追踪 Claude Code 的行为并把压缩后的记忆注入未来的会话。

这两者结合起来意味着什么？Claude Code 正在从"每次重新开始"走向"持续记忆 + 经验积累"的模式。这是一个质变，会让 AI 编程助手的上限大幅提升。

### Self-Evolving Agent：下一个方向

`lsdefine/GenericAgent`（872 stars）和 `EvoMap/evolver`（812 stars）都在做同一件事：让 Agent 自己演化技能树，而不是靠人工设计。GenericAgent 声称 6x Token 节省，EvoMap 用 GEP（基因组进化协议）驱动。

这个方向的竞争已经开始，下一步看哪家能真正跑通"不需要人工干预的自我改进闭环"。

### Vercel 入局：基础设施竞争加剧

`vercel-labs/open-agents`（738 stars）是 Vercel 开源的云端 Agent 部署模板。基础设施层的竞争和框架层的竞争同时在加速，门槛在快速提高。

### Google Magika：安静的实用工具

`google/magika`（854 stars）是一个用 AI 做文件类型检测的工具，零依赖，小而美。这种"把 AI 嵌入开发工具链日常环节"的用法比很多 fancy 应用更实在。

### 动手学 LLMs：中文社区在行动

`Lordog/dive-into-llms`（1385 stars）是《动手学大模型》系列的中文编程实践教程，配合沐神课程使用效果佳。对中文开发者来说，这是目前最系统的学习路径之一。
